# Mustafa Gasimzada

I'm a third-year Computer Engineering student at METU. Most of my time goes into systems that run on real hardware — embedded firmware, network infrastructure, low-level C.

At **METU ROVER** I built two firmware modules for a competition rover from scratch. One handled distributed wheel control — each wheel runs its own PID loop, manages its own safety logic, communicates over TCP independently. The other sat between the onboard computer and the power distribution board, switching subsystems via UDP. Both on RP2040s with hardwired Ethernet. I learned more from one field test gone wrong than from a semester of coursework.

Now at **WINS Lab** I'm building the edge infrastructure for an IoT botnet detection system — L2 bridge, kernel optimization, automated iptables mitigation. Working toward a publication at ACM ICIT 2026.
